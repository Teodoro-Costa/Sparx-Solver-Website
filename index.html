Sparx Solver

AI homework helper for Sparx Maths

Solve Sparx Maths questions with any AI you like.

Sparx Solver is a Chrome extension that reads a Sparx Maths question, sends it to the AI model of your choice, and shows a clean answer + explanation. Bring your own key from Groq, Google, or Anthropic — the extension is free and open to any model you plug in.

Install from Chrome Web Store Source on GitHub →

Features

Bring your own key

Groq (free), Gemini (free), or Anthropic Claude (paid). Nothing is billed by the extension.

Reads text and images

Handles image-based questions on supported vision models. Full markdown + LaTeX rendering.

Auto-fill answers

Optional: types the answer into the box for you. You choose whether to submit.

Bookwork tracking

Keeps a history of solved questions so you can review your workings later.

100 Club drill

Times-tables mode with a configurable minimum question time so answers look natural.

Local storage only

Your API key stays in your browser. No servers, no analytics.

How it works

  1. Install the extension from the Chrome Web Store.
  2. Add a free API key from Groq or Google AI Studio. Paste it into the popup.
  3. Open a Sparx question, click Solve or Auto-fill. Done.

FAQ

Is it free?

The extension is free. You pay only for the AI you choose — and Groq and Gemini offer free tiers that cover typical homework use.

Do you see my API key or my answers?

No. Everything runs locally in your browser. There is no Sparx Solver server. Your key is stored in Chrome local storage and only sent directly to the AI provider you pick.

Which AI is best?

Claude Sonnet 4.6 gives the most accurate maths answers if you're happy to pay. Groq's GPT-OSS is free and fast for text questions. Gemini Flash Latest is free and handles images.
